# Ejercicio-1-B-sico-API-de-promedio-
Desarrolla una API con Flask que calcule el promedio de un conjunto de calificaciones. Crea un endpoint POST /promedio que reciba un JSON con el nombre del estudiante y una lista de calificaciones. La API debe procesar los datos, calcular el promedio y devolver un JSON con el resultado.
